minTE.quickstart.seq_gen_demo.export_waveforms(seq, time_range=(0, numpy.inf))¶

Exports all waveforms from a seq object with associated timings for easy plotting

Parameters

time_range (iterable, default=(0, np.inf)) – Time range (x-axis limits) for all waveforms. Default is 0 to infinity (entire sequence).

Returns

all_waveforms – Dictionary containing all sequence waveforms and time array(s) The keys are listed here: ‘t_adc’ - ADC timing array [seconds] ‘t_rf’ - RF timing array [seconds] ‘’ ‘adc’ - ADC complex signal (amplitude=1, phase=adc phase) ‘rf’ - RF complex signal ‘gx’ - x gradient ‘gy’ - y gradient ‘gz’ - z gradient

Return type

dict
